MATHPARDY LIVE

Real-time multiplayer requires a free Firebase project.
This one-time setup takes about 5 minutes.

1
Create a Firebase Project

Go to console.firebase.google.com → Add project → Enter any name → Continue through setup.

2
Add a Web App & Copy Config

In your project → ⚙ Settings → "Your apps" → click </> (Web) → Register app → copy the firebaseConfig object.

3
Enable Realtime Database

Build → Realtime Database → Create database → Start in test mode → Done.

4
Enable Google Sign-In (optional)

Authentication → Sign-in method → Google → Enable → Save. (Skip to use username-only login.)

5
Paste Your Firebase Config Below

Paste the entire firebaseConfig = { ... } object or just the JSON object:

Config is saved locally in your browser. You only do this once per device.
